# App Screenshots

This directory contains screenshots of the EU Trip Tracker application for use on the landing page.

## Required Screenshots

1. **dashboard-screenshot.png** - Main dashboard view showing employee cards and compliance status
2. **employee-detail-screenshot.png** - Employee detail page with trip history and compliance calculations
3. **calendar-screenshot.png** - Calendar view showing trip timeline and safe re-entry dates

## Screenshot Guidelines

- **Dimensions**: 1200x800px minimum (16:10 aspect ratio)
- **Format**: PNG with transparency support
- **Quality**: High resolution, crisp text
- **Content**: Use realistic but anonymized data
- **Styling**: Match the current application theme

## How to Capture

1. Start the application locally
2. Add some sample employees and trips
3. Take screenshots of each key screen
4. Save with descriptive filenames
5. Optimize file sizes for web (under 500KB each)
